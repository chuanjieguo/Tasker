# Tasker
一个可爱的日程提醒器：让玲娜贝儿来提醒你~
# 设置主界面
包含WebTask、TipsTask、ExeTask三种任务，分别为定时抓取网络的指定内容并提醒、定时提醒文本、定时执行某可执行文件
